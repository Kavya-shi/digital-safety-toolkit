# 📱 Student Digital Safety Toolkit

## 📌 Overview
The Student Digital Safety Toolkit is a simple web-based project created to help students stay safe online. It spreads awareness about cyber threats like UPI fraud, phishing, and fake links.

---

## ❗ Problem Statement
Many students become victims of online scams because they:
- Share OTP unknowingly
- Click on fake links
- Trust unknown payment requests

This leads to financial loss and data theft.

---

## 💡 Solution
This project provides:
- Safety awareness tips
- Scam detection guidance
- Emergency help steps
- Interactive quiz

---

## 🎯 Features

### 🔐 Safety Tips
- Never share OTP
- Use strong passwords
- Avoid public Wi-Fi

### 🔍 Scam Detection
- Fake UPI requests
- Lottery scams
- Job frauds

### 🚨 Emergency Steps
- Block bank account
- Call helpline (1930)
- Report cybercrime

### 🧠 Quiz
- Test your knowledge
- Learn interactively
